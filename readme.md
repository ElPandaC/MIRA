# Project Mira

## Description
The goal of this project is to create an AI Agent capable of autonomous trading and predicting the behavior of the cryptocurrency market.

## Functionality
- **Autonomous Trading**: The AI Agent will be able to make independent decisions about buying and selling cryptocurrencies based on market data analysis.
- **Market Behavior Prediction**: Using machine learning and data analysis to forecast changes in the cryptocurrency market.

## Technologies
- **Programming Language**: Rust

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/ElPandaC/MIRA.git
2. Navigate to the project directory:
    ```bash

3.  Install the necessary dependencies:

# Проект Мира
## Описание
Целью данного проекта является создание AI Агента, способного на автономную торговлю и предсказание поведения крипторынка.

## Функциональность
- **Автономная торговля**: AI Агент будет способен самостоятельно принимать решения о покупке и продаже криптовалют на основе анализа рыночных данных.
- **Предсказание поведения рынка**: Использование машинного обучения и анализа данных для прогнозирования изменений на крипторынке.

## Технологии
- **Язык программирования**: Rust
